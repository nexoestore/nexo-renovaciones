# Nexo Renovaciones v1

Aplicación Android para controlar vencimientos y renovaciones de servicios de streaming.

## Funciones incluidas
- Dashboard con vencimientos de hoy, próximos 5 días y vencidos.
- Registro y edición de cliente, teléfono, servicio, valor, fecha y notas.
- Renovación rápida por un mes.
- Botón de WhatsApp con mensaje preparado.
- Alertas locales 5, 3 y 1 día antes, y el día del vencimiento.
- Reprogramación de alertas después de reiniciar el celular.
- Datos guardados localmente en el teléfono.
- Botón para probar una notificación.

## Compilar desde GitHub sin computador
1. Crea un repositorio nuevo en GitHub.
2. Sube todo el contenido de esta carpeta.
3. Abre la pestaña Actions.
4. Ejecuta el flujo "Construir APK Android".
5. Descarga el artefacto NexoRenovaciones-debug-apk.
6. Descomprime el archivo e instala el APK en Android.

## Primera prueba
1. Instala y abre la app.
2. Acepta el permiso de notificaciones.
3. Pulsa NUEVO y registra un servicio.
4. En INICIO, pulsa PROBAR NOTIFICACIÓN.
5. Verifica que aparezca la alerta.

## Nota
Esta primera versión no importa todavía archivos Excel. Los datos se registran manualmente para validar primero el diseño y el flujo de renovaciones.
